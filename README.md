# blue-app-neo-tests
Tests based on `demo.py` from https://github.com/coranos/blue-app-neo but using `@ledgerhq/hw-transport` instead.
Currently, only `node-hid` and `u2f` is supported.

## Prerequisites
Node >= 7.6.0

## Running the tests
```
npm install
```
```
npm run test:node-hid
```
```
npm run test:u2f
```
